# Bilder für die Landingpage

Diese beiden Fotos werden von `index.html` mit relativen Pfaden eingebunden.
Lege die Dateien hier ab (exakt diese Dateinamen), dann ist die Seite vollständig:

| Datei                 | Verwendung                          | Empfehlung                                  |
|-----------------------|-------------------------------------|---------------------------------------------|
| `hero-beratung.jpg`   | Hero-Hintergrundbild (Beratungsszene) | Querformat, ca. 1920×1280 px, JPG, optimiert |
| `portrait.jpg`        | „Über mich"-Porträtfoto              | Hochformat 4:5, ca. 800×1000 px, JPG         |

## Hinweis
Die Logos (Header + Footer) werden direkt von `https://change-manager.org/`
geladen und müssen **nicht** lokal abgelegt werden.

Das Hero-Overlay (dunkles Navy) ist bereits im Stylesheet (`.hero--with-image::after`)
hinterlegt, damit Headline und Button gut lesbar bleiben.
